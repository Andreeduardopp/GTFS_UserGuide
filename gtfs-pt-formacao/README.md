# GTFS PT Formação

> **O primeiro recurso educativo aberto em português sobre GTFS (General Transit Feed Specification).**

Dirigido a técnicos de operadores de transporte público, profissionais de sistemas de informação de mobilidade e estudantes de engenharia de transportes.

## Público-alvo

Profissionais e estudantes com:
- Conhecimento básico de ficheiros CSV e folhas de cálculo
- Alguma familiaridade com dados tabulares e bases de dados relacionais
- Necessidade prática de produzir, gerir ou consumir feeds GTFS

## Como Começar

1. Comece pelo [Módulo 0 — Antes de Começar](modulos/00_antes_de_comecar/README.md)
2. Siga a ordem numérica dos módulos
3. Pratique com os exercícios de cada módulo
