## [1.0.0] — 2026-05-03

### Adicionado
- Módulo 00 — Antes de Começar (história, importância, contexto europeu)
- Módulo 01 — Conceitos Base (modelo de dados, ficheiros, formato CSV, ZIP)
- Módulo 02 — Agência, Paragens e Rotas (agency.txt, stops.txt, routes.txt + exemplos STCP)
- Módulo 03 — Viagens, Horários e Geometria (trips.txt, stop_times.txt, shapes.txt, frequencies.txt)
- Módulo 04 — Calendários e Exceções (calendar.txt, calendar_dates.txt, feriados portugueses)
- Módulo 05 — Acessibilidade e Transferências (wheelchair_boarding, transfers.txt, pathways.txt, levels.txt)
- Módulo 06 — Ciclo de Vida dos Dados (fontes, ETL, validação, publicação, qualidade)
- Módulo 07 — Tarifas e Extensões (Fares v1/v2, GTFS-Flex, GTFS-ride, Andante Porto)
- Módulo 08 — Ferramentas Práticas (validador, Python/partridge, visualização, pipeline ETL)
- Exercícios para módulos 01–07 com contexto português (CP, Carris Metropolitana, STCP, TransBeira)
- Exemplos CSV reais e sintéticos para todos os módulos
- Recursos: glossário, links úteis, comparação GTFS vs NeTEx
- Registo central de referências (STD, BP, REG, PT, TOOL, DATA)
- Rede de exemplo: STCP Porto Linha 200 (dados reais)
