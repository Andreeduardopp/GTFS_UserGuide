# Módulo 7 — Perfil NeTEx Portugal

> *Tempo estimado: 1,5 semanas*

## Objetivos de Aprendizagem

- Compreender o que é um perfil nacional e porque Portugal tem o seu
- Consultar o perfil em `ptprofiles.azurewebsites.net`
- Distinguir campos obrigatórios vs opcionais no contexto português
- Submeter dados ao NAP nacional (`nap-portugal.imt-ip.pt`)

> **Este módulo é único em todo o ecossistema de recursos NeTEx** — não existe nada equivalente em nenhuma língua sobre o perfil português.
